# Mario Bros with libGDX

Platform game like Super Mario Bros 1 using libGDX Library


## Prerequires

1. Git 2.6+
2. Gradle 2.11+
3. Java 8+


## How to Play

Clone

```
git clone https://github.com/humbertodias/game-mario-bros-libgdx.git
```

Inside the folder

```
cd game-mario-bros-libgdx/src
```

Run

```
gradle desktop:run
```
